rfind

类似于find()函数，不过是从右边开始查找的

  • 用法： 变量.rfind(“str”)

find

检测字符串（str）是否包含在变量（）中，如果是，返回开始的索引值，否则返回-1

  • 用法：变量.find（”str”）

index

检测字符串（str）是否包含在变量（）中，如果是，返回开始的索引值，否则报错 （可加范围）

正则匹配

字符 功能
. 匹配任意一个字符（除了\n）
[] 匹配[ ]中列举的任意字符
[^…] 匹配 除了字符组中字符 的 所有字符
\d 匹配数字，即（0-9）
\D 匹配非数字，即 不是数字
\s 匹配空白字符 【\t(横向制表符)\n(换行)\r(回车)\f(换页)】
\S 匹配非空白字符
\w 匹配单词字符 a-z A-Z 0-9 _
\W 匹配非单词字符
***** 匹配前一个字符出现 0 次或无限次，即 可有可无
+ 匹配前一个字符出现 1 次或无限次，即 最少有一次
？ 匹配前一个字符出现 0 次 或 1 次 ，即 要么一次，要么没有
{m} 匹配前一个字符出现 m 次
{m,n} 匹配前一个字符出现从 m 到 n 次
^ 匹配字符串开头
$ 匹配字符串结尾
